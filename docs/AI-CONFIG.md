# AI-CONFIG.md — المصدر الموحّد للحقيقة (Single Source of Truth)

> هذا الملف هو **أهم ملف في المشروع** لأنظمة الذكاء الاصطناعي.
> اقرأ هذا الملف أولاً في كل جلسة عمل قبل أي تعديل.
> لا تعتمد على أي ملف توثيقي آخر دون التحقق من هذا الملف أولاً.

---

## 1. ما هو هذا المشروع؟

**AUHMC 2026** — منصة رقمية متكاملة للمؤتمر العلمي الأول لمشفى حلب الجامعي.
موقع **Full-Stack** مبني بـ Node.js + Express + SQLite + Vanilla JS (لا يوجد Framework).

## 6. قواعد التعديل الآمن

### 1. الملفات طويلة — استخدم البحث الدقيق
- `styles.css` حوالي 900 سطر
- `app.js` حوالي 250 سطر
- `admin.js` حوالي 450 سطر
- استخدم `replace_in_file` مع SEARCH/REPLACE دقيق

### 2. لا تعيد تنسيق الملفات
- لا تقم بتجميل أو إعادة تنسيق CSS أو JS
- استخدم التعديلات المستهدفة فقط

### 3. جميع البيانات في قاعدة البيانات — وليس في الكود
- `src/frontend/app.js` يقرأ من API (`/api/content`)
- `src/frontend/admin.js` يقرأ ويكتب عبر API
- البيانات الافتراضية تُبذر تلقائياً في SQLite (في `src/server/database.js`)

### 4. الأيقونات من Font Awesome
- جميع الأيقونات تستخدم Font Awesome 6.5.1

### 5. الألوان عبر متغيرات CSS
- استخدم `var(--primary)`, `var(--gold)`, إلخ
- المتغيرات معرفة في `:root` في `styles.css`

### 6. الاستجابة (Responsive)
- التصميم Mobile-First
- نقاط القطع: 992px, 768px, 480px

### 7. اللغة
- جميع النصوص الظاهرة للمستخدم بالعربية
- الكود (المتغيرات، التعليقات) بالإنجليزية
- المصطلحات الطبية باللاتينية بين قوسين
